<h1 align="center">SI-Magang — Sistem Informasi Manajemen Magang</h1>

<p align="center">
  Sistem modern berbasis web untuk mengelola proses magang secara terpusat: presensi, aktivitas harian, progress, penilaian, hingga laporan akhir.
</p>

---

## Fitur Utama

- **Manajemen Role** (Admin, Mahasiswa, Mentor, VP)
- **Presensi QR Code** + riwayat kehadiran
- **Aktivitas Harian & Progress Mingguan**
- **Penilaian Kinerja Mahasiswa**
- **Manajemen Dokumen & Laporan Magang**
- **Dashboard Statistik Real-time**

---

## Teknologi yang Digunakan

| Layer | Teknologi |
|------|------------|
| Backend | Laravel |
| Frontend | Blade + Tailwind CSS |
| Database | MySQL |
| Library | Spatie Permission, Chart.js, QR Code Generator |
